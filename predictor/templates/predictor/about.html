О проекте

Arctic Ice Predictor

Прогнозирование таяния арктического льда с помощью нейронных сетей

Проект использует глубокое обучение для анализа и прогнозирования состояния морского льда в Арктике. Модель основана на архитектуре ConvLSTM, специально разработанной для обработки пространственно-временных данных.

🎯 Цели проекта
  • Мониторинг изменений в арктическом регионе
  • Прогнозирование таяния льда на 30-90 дней вперед
  • Исследование влияния климатических факторов
  • Предоставление данных для научных исследований
🧠 Архитектура нейросети
ConvLSTM Model:
├── ConvLSTM1 (7 → 16) с ядром 3x3
├── ConvLSTM2 (16 → 32) с ядром 3x3
├── ConvLSTM3 (32 → 16) с ядром 3x3
└── Decoder (16+1 → 1)
    ├── Conv2D (16+1 → 32)
    ├── ReLU
    ├── Conv2D (32 → 16)
    ├── ReLU
    └── Conv2D (16 → 1) + Sigmoid
📊 Входные данные
  • Концентрация морского льда
  • Температура поверхности
  • Поле ветра (u-компонента)
  • Поле ветра (v-компонента)
  • Океанические течения (u)
  • Океанические течения (v)
  • Батиметрия (глубина)
🔬 Технологии
Python 3.9 Django PyTorch Lightning NumPy Matplotlib Bootstrap 5
📈 Метрики качества
  • MSE (Mean Squared Error) - среднеквадратичная ошибка
  • MAE (Mean Absolute Error) - средняя абсолютная ошибка
  • RMSE (Root Mean Squared Error) - корень из MSE
